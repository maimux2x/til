# bin/setup を読んだメモ

## ざっくり何をしているファイルか

Campfire の開発環境をセットアップするための Ruby スクリプト

## 読んで分かったこと

Homebrew, packman, aptなどのパッケージマネージャーを検出して、必要なパッケージをインストールするようになっている

- sqlite
- ffmpe
- mise

## 気になった/学びになったポイント

28行目で production の場合は、スクリプトの実行を中断するように実装されている

bin/setup にはデータベースとファイルストレージを削除する処理、ログとtmpを削除する処理、パッケージのインストールが含まれているため、production で実行してしまうと大切なデータの消失やシステムの状態を変更してしまうリスクがある
