## Arch Linux パッケージ更新

1. パッケージデータベースを更新

```
$ sudo pacman -Sy
```

このコマンドで最新のパッケージ情報を取得する。

2. システム全体をアップデート

```
$ sudo pacman -Syu
```

`-S` パッケージ操作
`-y` パッケージデータベースの更新
`-u` アップグレード
ステップ１のコマンド単体で実行せず、`-Syu` でまとめるのが一般的？

 Arch Linux はローリングリリースなので、必ずシステム全体を更新すること

3. キャッシュを整理

```
$ sudo pacman -Sc
```

### お役立ちコマンド

更新されるパッケージの確認

```
$ pacman -Qu
```

孤立パッケージ（依存関係がなくなったもの）の確認

```
$ packman -Qdt
```

孤立パッケージの削除

```
$ sudo packman -Rns $(pacman -Qdtq)
```
